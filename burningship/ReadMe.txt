You can move across the complex plane with arrow keys.
To zoom in the fractal use left shift key, to zoom out, left ctrl key.

The default fractal in exe is burning ship fractal, but if you understand glsl shading programming, you can change the equation in fractal.fragmentshader which is pretty much like c only to get any fractals.

The opengl code is made to run on nvidia graphics cards. So, it will give best performance if you have nvidia gpu.
